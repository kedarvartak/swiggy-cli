# Swiggy Workspace

This repository is now organized as a small multi-app workspace:

- `cli/` - existing TypeScript CLI
- `frontend/` - React frontend scaffold
- `backend/` - Python backend scaffold
- `developer_docs/` - shared Swiggy product and API documentation
- `docs/` - shared architecture and roadmap notes

The long-term direction is to keep workflow and integration logic shared between the CLI and UI through the backend, while preserving the CLI as a first-class interface.
